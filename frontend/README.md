# Sentinel Fusion AI Frontend

Enterprise React frontend for Sentinel Fusion AI.

## Stack

- Next.js 15
- React 19
- TypeScript
- TailwindCSS
- TanStack Query
- Zustand
- React Hook Form
- Zod
- Axios
- Framer Motion

## Scripts

- `npm run dev`
- `npm run build`
- `npm run test`
- `npm run typecheck`
