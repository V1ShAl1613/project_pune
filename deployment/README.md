# Sentinel Fusion AI Deployment

This directory contains the production deployment entry points for Sentinel Fusion AI.

Included layers:

- Docker runtime hardening and compose profiles
- Helm chart for app deployment
- Kubernetes platform manifests
- CI/CD workflow definitions
- Observability, logging, backup, restore, and release assets

The application code is unchanged by these artifacts. They package the existing FastAPI and Next.js services for production use.
